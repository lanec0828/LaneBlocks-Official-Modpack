# LaneBlocks Official Modpack
### this is the official LaneBlocks Minecraft Server Modpack, built for performance and Compatability with our Unique Rules

# Installation Guide

## Prism Launcher

### 1) Install Prism Launcher
- download the latest Prism Launcher for your OS from https://prismlauncher.org/download/ and install the app.

### 2) Import the modpack into Prism Launcher 
- Download The .Zip File From `Releases`
- Drag The .Zip into Prism Launcher

### Troubleshooting

- If the instance fails to start, try to restart Prism Launcher or Update If applicable
- Ensure you are connected to the interwebs
- If you see crashes, plaese report at the Issues Tab
